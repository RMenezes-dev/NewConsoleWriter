# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog,
and this project adheres to Semantic Versioning.

## [Unreleased]

### Added
- Support for simple headers with customizable colors and text transformation.

## [1.0.0] - 2026-01-19

### Added
- Core methods for writing colored text to the console.
- Support for foreground and background colors.
- Semantic methods: WriteSuccess, WriteWarning, WriteError and WriteInfo.
- Optional text transformations: UpperCase, LowerCase and TitleCase.
- Automatic restoration of console colors after writing.
